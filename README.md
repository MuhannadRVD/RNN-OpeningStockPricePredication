📈 Google Stock Price Prediction using RNN + LSTM 🚀

This project uses Deep Learning (RNN + LSTM) to predict the opening stock price of Google (GOOGL) based on historical stock market data over a selected time period.

The model learns patterns from previous stock prices and predicts future opening prices with high accuracy. It also includes a visualization comparing the real stock prices vs predicted stock prices using line charts for better performance analysis.

🔥 Features
📊 Historical Google stock price analysis
🧠 RNN + LSTM Deep Learning model
📈 Open price prediction
📉 Real vs Predicted stock price visualization
🗂 Data preprocessing and normalization
⚡ TensorFlow / Keras implementation
🛠 Technologies Used
Python 🐍
TensorFlow / Keras
NumPy
Pandas
Matplotlib
Scikit-learn
🎯 Project Goal

To build an AI model capable of learning stock market behavior and predicting future opening prices based on past trends.
